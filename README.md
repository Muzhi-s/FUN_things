# 🛠️ FUN Things - 实用小工具集锦

这里是我自己开发和维护的一些实用小工具集合，旨在解决日常工作和生活中的一些痛点，同时也用于技术探索和实验。项目会持续更新中...

## 📂 项目列表

### 1. ⏱️ 自动关机工具 (Auto Shutdown Tool)
**目录**: `anto_shutdown/`
- **简介**: 一个基于 Python 开发的 Windows 命令行工具，支持定时关机、倒计时关机以及重启功能。
- **场景**: 适合下载大文件挂机、睡前听歌自动关机等场景。
- **详情**: [查看文档](./anto_shutdown/README.md)

### 2. 📑 PDF 中文水印工具 (PDF Watermark)
**目录**: `pdf_add_watermask/`
- **简介**: 专注于为 PDF 文件添加平铺式文字水印的 Python 脚本。
- **特色**: 
    - 自动识别并使用系统中文字体（解决乱码问题）。
    - 支持自定义旋转角度、透明度、间距和颜色。
- **详情**: [查看文档](./pdf_add_watermask/README.md)

### 3. 🎨 Tailwind CSS 实验场 (Tailwind Project)
**目录**: `tailwind-project/`
- **简介**: 用于学习和测试 Tailwind CSS 样式的 Web 前端项目。
- **技术栈**: Vite + React + Tailwind CSS。

---

## 🚀 如何使用

每个工具都在各自的子目录中独立维护，请进入相应的目录查看具体的 `README.md` 说明文档以获取安装和运行指南。
